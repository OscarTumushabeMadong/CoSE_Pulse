# Contributing

Thank you for your interest in contributing.

## Coding Standards

- Write modular code.
- Document all functions.
- Keep parsers department-specific.
- Use descriptive variable names.

## Workflow

1. Create a branch.
2. Implement your feature.
3. Test thoroughly.
4. Submit a pull request.

## Philosophy

Every improvement should make CoSE Pulse more accurate, maintainable, and useful for students.
